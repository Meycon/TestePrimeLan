# TestePrimeLan
**Teste para vaga de desenvolvedor Android - PrimeLan **

 Criar um aplicativo Android nativo de consulta de CEP, o aplicativo deverá consultar um serviço REST público de CEP (https://viacep.com.br/)

- A interface deverá conter um campo para digitar o CEP e um botão de consulta.
- Caso o CEP seja inválido avisar ao usuário
- Seguir style guide do material design
- Utilizar o pattern Android Architecture Components
